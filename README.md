# charts

a lightweight canvas chart library


## Quickstart

```
$ git clone https://github.com/wineSu/charts.git
$ cd charts
$ npm i
$ npm run dev
$ npm run build
```

## Usage

```
const lineChart = Charts({
    id:'can1',
    type:'line',
    data:[{
        xVal:'5-26',
        yVal:40
    },{
        xVal:'5-27',
        yVal:20
    },{
        xVal:'5-28',
        yVal:30
    },{
        xVal:'5-29',
        yVal:80
    },{
        xVal:'5-30',
        yVal:60
    },{
        xVal:'6-01',
        yVal:20
    }]
});

... 

```
